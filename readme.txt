Sexy Spike3 - 3D Volley Ball Game - v0.2

Developer: Hyokyung Lee. Woongjae Lee. Tom Jaejoon Lee
E-mail: serialkk@naver.com
Developement Environment: VisualStudio 2008 + DirectX 2003 Summer

조작키:
 키보드 방향키 + 'Z':리시브, 토스, 'X': 스파이크, 'V':네트에 공이 걸리면 공을 빼냅니다.
 공이 빨간색일때 키를 누르면 캐릭터가 알아서 공을 치게 되있습니다.
 프레임 수가 낮으면 게임이 제대로 동작하지 않을 수 있습니다. 권장 프레임은 60이상입니다.

권장사양:
 CPU: 인텔 듀얼코어 2.8Ghz
 메모리: 게임실행시 500메가의 메모리를 점유하므로 500메가 이상의 메모리를 가진 시스템
 비디오카드: 지포스 9500GT이상
